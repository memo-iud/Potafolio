# **App Name**: GLM Portfolio

## Core Features:

- SEO Metadata: Configures SEO-friendly title and meta description to enhance search engine visibility.
- Hero Section: Displays a concise introduction with a title, subtitle, and brief paragraph, along with clear calls to action.
- About Me Section: Provides a personal overview, highlighting background, passion for technology, and a personal motto.
- Project Showcase: Highlights key projects with descriptions, technologies used, and links to GitHub repositories or live demos.
- Skills Overview: Presents a visual summary of technical skills, including programming languages, frameworks, and databases.
- Contact Information: Includes contact details (email, LinkedIn, location) and a direct call to action to encourage inquiries.

## Style Guidelines:

- Primary color: Deep blue (#293B5F) to convey professionalism and trustworthiness.
- Background color: Light gray (#F0F0F0) for a clean, modern backdrop.
- Accent color: Teal (#3D5A80) for calls to action and interactive elements, providing a subtle contrast.
- Headline font: 'Space Grotesk' (sans-serif) for a tech-forward, modern feel.
- Body font: 'Inter' (sans-serif) for readability and a clean aesthetic.
- Simple, outline-style icons to represent skills and technologies, maintaining a clean interface.
- A clean, grid-based layout to organize projects and skills effectively.
